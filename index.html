1. Authentication

iTRUST PAYMENT GATEWAY offers two forms of authentication to ensure secure and seamless integration:

i. Bearer Authentication

This method utilizes an open protocol to securely authorize access to our services in a simple and standardized way across web, mobile, and desktop applications.

ii. Bearer Token

The Bearer Token is a JWT (JSON Web Token) that you receive after providing your application Name, Client ID, and Client Secret.

This flexible authentication mechanism ensures that only authorized applications can interact with our API securely.
